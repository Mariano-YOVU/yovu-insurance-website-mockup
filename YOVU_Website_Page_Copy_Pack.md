# YOVU Website Page Copy Pack

Working copy and page-design notes for the recommended launch sitemap.

Use each expandable section as a working page brief for copywriting, wireframing, and design.

## How to use this document

- Each section includes the page goal, target audience, hero copy, section-by-section copy, and layout or graphic guidance.
- Copy is written to be strong enough for draft page builds, with room for normal design and legal review before launch.
- Insurance is the lead story across the site, but non-insurance visitors should always have a clear path forward.
